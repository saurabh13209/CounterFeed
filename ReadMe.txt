As 
the name says “CounterFeed” The app is focused on revealing the authenticity of the news and online content.
The
main features are polling, comments which are obtained by the users and this
will help determine if the news is real or fake.
Any user can upload an image,
description, link or location of the news and can post that in the public feed,
while if any user knows if it’s a real or fake, they can vote it and make a
comment. 
There is also a login system in the app in which the history of the
user is stored, it also has an anonymously news uploading feature.
 If any news
gets too many hits and it will appear in the trending section.
